🔑 GitHub設定（初回だけ）

トークンの取得方法：
GitHubにログイン → 右上のアイコン → Settings → 一番下の「Developer settings」→「Personal access tokens」→「Tokens (classic)」→「Generate new token」
✅ repo にチェックを入れて生成 → 表示されたトークンをコピーしてここに貼り付け
